The Code for CENTI-SPRINTER.


                    
//get the value from the Page
//starts or controller function
function getValues(){
// get value from page
let startValue = document.getElementById("startValue").value;
let endValue = document.getElementById("endValue").value;

//parse into integers
startValue = parseInt(startValue);
endValue = parseInt(endValue);

if(Number.isInteger(startValue) && Number.isInteger(endValue)){

//we call generateNumbers
let numbers = generateNumbers(startValue,endValue);
//call displayNumbers
displayNumbers(numbers);

} else {
    alert("You must enter integers");
}

}

//generate numbers from the startValue to the endValue
//logic function
function generateNumbers(sValue, eValue){

 let numbers = [];

 //we want to get all number from start to end
for(let index = sValue; index<= eValue; index++){
    //this will execute in a loop until index equals eValue
    numbers.push(index);
}

 return numbers;

}

//display the numbers and mark the even numbers bold
//disaply or view function
function displayNumbers(numbers) {

    let templateRows="";

    for (let index = 0; index < numbers.length; index++) {
        let className = "even";
        let number = numbers[index];

        if(number % 2 == 0){
            className = "even";
        }
        else{
            className = "odd";
        }
        //This does not render correctly with Prism. 
        //See the source code.
        templateRows += `${number}`;

    }
document.getElementById("results").innerHTML = templateRows
}

The Code is structured in three functions.

FUNCTION: "getValues"

    The first 2 'let' statements retrieve the values entered from the HTML page and places them into variables. Next the variables are parsed into integers.

    The 'if' statement checks one last time the at the variables are integers before passing them on to the 'generateNumbers' function. If they fail, a message is sent to the user that they must enter integers into the HTML page.

    After the numbers are generated by the 'generateNumbers' function, they are then passed on to the 'displayNumbers function.

FUNCTION: "generateNumbers"

    First the function receives variables from the 'getValues' function and stores them into sSValue, and eValue.

    Then an array is defined called numbers. Then we run a loop to obtain all the numbers from the start and end Values. The result is passed back to the 'getValues' function.

FUNCTION: "displayNumbers"

    The functions starts by receiving the number list that was created by the 'generateNumbers' function, and then passed back to the 'getValues' function before being sent to this function.

    a new variable is created to hold the display formatting for the table on the HTML page to hold the results. Next we check each number passed through to see if it's even or odd and assign it a classname.

    As each number is processed, a table row of html is created and appended to the 'templateRows" variable. Once the final row has been calculated, all of the generatd html code is inserted into the HTML page as a tale in the results area.
